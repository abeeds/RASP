Functions
       
connect_db()
This function connects to a MongoDB database named RASP. If the environment variable named CLOUD_MONGO is set to 0, it connects to a locally running database. If CLOUD_MONGO is set to 1, it connects to the cloud db. The password for the CLOUD db is set in GAME_MONGO_PW.
       
insert_one(collection, doc, db=USER_DB)
Inserts a document into the specified collection. The db by default is set to RASP.
       
fetch_one(collection, filt, db=USER_DB)
Searches database at a specific collection with a filter, filt, and returns the first doc found.
       
fetch_all(collection, db=USER_DB)
Finds all documents in the specified collection with a filter.
       
fetch_all_as_dict(key, collection, db=USER_DB)
returns all documents in a collection as a dict.
       
update_one(collection, filt, doc, db=USER_DB)
Updates a doc in the specified collection using a filter.
